# Sistema de Predicción de Asistencia a Museos

Aplicación desarrollada en Streamlit para el despliegue productivo de un modelo predictivo orientado a estimar la asistencia mensual a museos.

## Descripción

El proyecto permite visualizar la evolución histórica de visitantes, comparar modelos predictivos y realizar predicciones en vivo mediante un modelo XGBoost previamente entrenado en Google Colab.

## Funcionalidades

- Análisis histórico de asistencia mensual.
- Predicción en vivo con nuevos registros.
- Comparación de modelos predictivos.
- Visualización de valores reales vs predichos.
- Conclusiones y limitaciones del proyecto.

## Modelo utilizado

El modelo seleccionado para el despliegue fue XGBoost, debido a su desempeño global en la comparación de métricas, mostrando mas rapidez y eficiencia pero sacrificando interpretabilidad
